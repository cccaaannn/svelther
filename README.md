### Running
- Get a api key from [openweathermap](https://openweathermap.org/)
- Use `.env.template` file to create a `.env` file
- Add your api key to `VITE_OPENWEATHERMAP_API_KEY` field
- Development
    - Run `npm run dev`
- Deployment
    - Run `npm run deploy`
